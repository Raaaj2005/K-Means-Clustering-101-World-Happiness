# 🌍 K - Means Clustering 101: World Happiness

## 📖 Overview
This project focuses on applying unsupervised machine learning techniques to perform geographical data segmentation. By processing image datasets and utilizing clustering algorithms, the project identifies distinct spatial patterns and visualizes complex geographical data effectively.

## 🎯 Objective
The primary objective is to leverage the K-Means clustering algorithm to perform unsupervised segmentation on geographical data. This involves finding the optimal number of clusters and creating interactive visual representations of the segmented spatial data.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Environment:** Jupyter Notebook
* **Machine Learning:** Scikit-Learn
* **Visualization:** Plotly

## 🧠 Key Learnings & Skills Demonstrated
* **Geographical Visualization:** Using Plotly to create interactive visualizations of spatial data.
* **Image Processing:** Importing and visualizing dataset images using Python libraries.
* **Algorithm Intuition:** Understanding the mathematical theory and practical application behind the K-Means clustering algorithm.
* **Cluster Optimization:** Applying the Elbow Method via Scikit-Learn to algorithmically determine the optimal number of data clusters.
* **Model Implementation:** Executing K-Means in Scikit-Learn to perform the final unsupervised segmentation.

## 🚀 Getting Started

### 📋 Prerequisites
To run this project, you will need:
* Python 3.x
* Jupyter Notebook or JupyterLab

### ⚙️ Installation & Setup

1. Clone the repository to your local machine:
   git clone https://github.com/Raaaj2005/your-repo-name.git

2. Navigate into the downloaded project folder:
   cd your-repo-name

3. Install the required dependencies:
   pip install scikit-learn plotly jupyter

4. Launch the Jupyter Notebook environment:
   jupyter notebook

5. Open the project's .ipynb file and execute the cells sequentially.
